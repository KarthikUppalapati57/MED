Labor Workflow Artifact Time Clock and Tip Pooling operational flows

Live workflow map for labor capture and tip distribution.

This artifact documents the current product behavior, the intended operator flow, and the wiring gaps that need to be closed before the modules behave end to end in production.

Partially Wired

Implementation Status

Time Clock can write punches to Supabase. Tip Pooling has the distribution UI and math, but needs API registration, schema field alignment, and real date filtering.

Source Data time_clocks

Stores organization, location, employee profile, clock-in time, clock-out time, and punch status.

Distribution Rule Hours-Based Split

Each eligible employee receives tips proportional to completed hours worked in the selected period.

Calculate Split

The app totals labor hours, calculates each employee ratio, and multiplies that ratio by total tips.

Distribution Formula

employee_hours = clock_out_time - clock_in_time total_hours = sum(employee_hours) employee_ratio = employee_hours / total_hours tip_share = total_credit_card_tips * employee_ratio
